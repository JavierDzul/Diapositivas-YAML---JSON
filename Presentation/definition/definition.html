Introducción

Definición

Una interrupción

es...

  • Un conjunto de funciones.
  • Estas funciones facilitan la programación de aplicaciones.
  • Indícan una operación especifica al sistema.
  • Pueden ser de software o hardware.

Definición

Una interrupción

se invoca...

Mediante la sintaxis:
INT value.

Value toma valores desde 0 a 255.

Tiene subfunciones que permiten especificar aún más.

Las subfunciones se especifican con el registro AH : 0 a 255.

Por lo tanto (255 * 255) = 65536 funciones.

Otros registros pueden ser usados también para parametrar.
