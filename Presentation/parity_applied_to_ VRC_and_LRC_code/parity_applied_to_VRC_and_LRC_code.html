Forma básica de detección de errores, incorpora un bit de paridad al dato transmitido.
Bit de Paridad

Un bit que toma un valor binario (0 ó 1) para satisfacer una restricción de paridad sobre un determinado patrón de bits.

Es posible implementar un esquema de paridad par o impar.

Un patrón de bits se considera par o impar si y sólo si tiene un peso par o impar respectivamente.

Bit de Paridad

Es posible definir dos caminos detección de errores con el bit de paridad:

  • Código de paridad par: se incorpora un bit adicional.
  • Código de paridad Impar: se incorpora un bit adicional.

Por convención, el bit de paridad lo indicaremos a la izquierda del patrón de bits original.
