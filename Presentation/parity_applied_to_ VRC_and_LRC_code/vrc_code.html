Vertical Redundancy Check

El código VRC añade un bit de paridad a cada unidad de datos de manera que la cantidad de bits en 1 sea par o impar.

  • Bit de paridad para cada unidad, en vez de uno para todo el mensaje.
  • Detecta la totalidad de los errores SIMPLES a nivel de unidad de datos
